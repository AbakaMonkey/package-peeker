# Package Peeker

A lightweight minimalist extension for `pacman` that organizes all your packages in an organized list and allows easy access to remove them

## Installing
```
git clone https://github.com/AbakaMonkey/package-peeker
cd package-peeker
pipx install .
```
## To run the program:
`package-peeker`

This DOES require the user to be root, but the script does that for you. And also I just don't want to figure out why appending sudo before the command just doesn't work... JUST LET IT DO IT FOR YOU MKAY?
